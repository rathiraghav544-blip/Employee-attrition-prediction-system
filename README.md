# Employee Attrition Prediction System

An AI-powered HR Analytics web application that predicts employee attrition risk using Logistic Regression and Streamlit.

## Features

- Predicts employee attrition risk
- Probability-based prediction
- Interactive Streamlit dashboard
- HR insights generation
- Real-time prediction system

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Logistic Regression

## Machine Learning Workflow

1. Data Cleaning
2. Exploratory Data Analysis (EDA)
3. Feature Engineering
4. Data Scaling
5. Logistic Regression Model Training
6. Model Evaluation
7. Streamlit Deployment

## Features Used

- Age
- Monthly Income
- Years At Company
- Job Satisfaction
- OverTime

## Model Performance

- Logistic Regression
- Balanced Class Weights
- Accuracy: ~89%

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
